Nada English Academy V21.1.1

تم إصلاح مشكلة إعادة تحميل الصفحة بشكل متكرر.
السبب كان تحميل بيانات Firebase تلقائيًا بعد كل إعادة فتح، ثم تنفيذ reload، مما كان يعيد نفس العملية بلا نهاية.

التعديل:
- تحميل بيانات الحساب تلقائيًا مرة واحدة فقط خلال جلسة المتصفح.
- استمرار المزامنة التلقائية بدون تكرار تحديث الصفحة.
- تحديث Cache الخاص بـ Service Worker لضمان تحميل الملفات الجديدة.

بعد رفع النسخة على GitHub Pages:
1) انتظري دقيقة.
2) افتحي الموقع واضغطي Ctrl + F5 مرة واحدة.
