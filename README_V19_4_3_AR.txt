Nada English Academy - نسخة الموبايل PWA

التثبيت على Android:
1) ارفعي محتويات هذا المجلد على استضافة HTTPS مثل Netlify أو GitHub Pages.
2) افتحي الرابط في Google Chrome على الموبايل.
3) اضغطي زر "تثبيت" داخل التطبيق أو من قائمة Chrome اختاري "إضافة إلى الشاشة الرئيسية".
4) بعد أول فتح، معظم أجزاء التطبيق تعمل بدون إنترنت.

مهم:
- لا تفتحي index.html مباشرة من مدير الملفات؛ التثبيت يحتاج رابط HTTPS.
- بيانات التقدم والمواضيع المضافة محفوظة في Local Storage على نفس المتصفح والجهاز.
- استخدمي قسم التصدير لعمل نسخة احتياطية قبل مسح بيانات المتصفح أو تغيير الهاتف.
